# Project Quản Lý Kho Siêu Thị



### Prerequisites

Yêu cầu trước khi cài đặt phần mềm

```
mysql 5.7.x && java 8.x
```

### Installing

Tải Source code về cài đặt

```
git clone https://github.com/phamthanhnghia/QuanLyKhoSieuThi
```
